## Requirements

- Installed Docker
- Installed node

## Environment

- PHP 8.1
- Laravel 9.5
- Mysql MariaDB 10.8
---
- Nodejs 16.13.0
- Npm 8.1.0
- Vuejs 3.2.31
- Vue Router 4.0.13

## Setup

```
docker-compose build
```

## Run

```
docker-compose up
```

```
cp .env.example .env
```

```
npm install
```

```
npm run hot
```

## URL

| Name  | URL |
| ------------- | ------------- |
| App | https://localhost/

## Shutdown

```
docker-compose down
```
